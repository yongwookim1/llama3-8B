# Run

### Download Ollama

https://ollama.com

### Clone repository

```bash
git clone https://github.com/yongwookim1/llama3-8B.git .
cd ollama-modelfile/Llama-3-8B-Instruct
```

### Create llama3-8B

```bash
ollama create llama3-instruct-8b -f Modelfile
```

### Ollama model list

```bash
ollama list
```

### Run llama3-8B

```bash
ollama run llama3-instruct-8b
```
